1. General Configuration

The top-level parameters configure the overall behavior of the system, including model selection and experiment settings.

Parameter Value Description
model_name "gpt2" The foundation model being used for this experiment. GPT-2 is a transformer-based language model developed by OpenAI, capable of generating coherent text.
model_dir "" Directory where the model files are stored. An empty string indicates that the model will be loaded from the default location or using the HuggingFace library.
experiment_dir "./" Directory where experiment outputs, logs, and saved models will be stored. The current directory ("./" or "./") is used here.
problem "clustering" The optimization problem being solved. The configuration is set up for clustering by default, but also contains parameters for MST and line scheduling.
algorithm "all" Indicates which algorithms to run. The value "all" means that all implemented algorithms (best_of_llm, best_of_alg, top_ift) will be executed and compared.

3. Problem Parameters

These parameters define the specific characteristics of each optimization problem.

3.1 Clustering

Clustering involves grouping a set of points so that points in the same group are more similar to each other than to those in other groups.

Parameter Value Description
num_points 24 Number of data points to be clustered in each problem instance.
r 10 Radius parameter that affects the spatial distribution of clusters, potentially determining the maximum distance between points in a cluster.
num_test_points 50 Number of test instances to generate for evaluating the clustering performance.

3.2 Minimum Spanning Tree (MST)

The Minimum Spanning Tree problem aims to find a subset of edges that forms a tree including every vertex, where the total weight of all the edges is minimized.

Parameter Value Description
num_lines 4 Number of lines or paths in the graph structure.
prob 0.4 Probability parameter used in graph generation, likely controlling edge density or connection probability.
deg 2 Degree parameter that may specify the average or maximum vertex degree in the graph.
num_test_points 50 Number of test instances to generate for evaluating MST algorithm performance.
